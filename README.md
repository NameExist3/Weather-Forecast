# Weather-Forecast
使用java,xml等编写简单的天气预报App安卓项目
- 实现的功能有：
  1. 城市的天气预报以及未来三天的天气情况
  2. 城市存储在数据库实现增删功能
  3. 清理缓存等等
  
- 使用的IDE工具：Intellij IDEA
- 使用的框架有：
  1. Picasso：加载网络图片的框架
  2. xutils：封装完善的多功能框架，此处主要使用网络获取数据功能
  3. Gson：解析json数据的框架，简易方便
